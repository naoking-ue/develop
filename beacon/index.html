Beacon Sample

Available in Chrome 39+

Beacon lets you asynchronously transfer small HTTP data from the User Agent to a web server. It can be used e.g. to send analytics or diagnostics code without delaying the page's unload or affecting the performance of the next navigation.

The sample below sends a beacon to a sample server at https://putsreq.herokuapp.com/Dt7t2QzUkG18aDTMMcop on unload. Close this page and visit PutsReq inspect page to see if the beacon data was received. 

      
      function() {
          window.addEventListener('pagehide', logData, false);
          function logData() {
              navigator.sendBeacon("https://putsreq.herokuapp.com/Dt7t2QzUkG18aDTMMcop", 'Sent by a beacon!');
          }
      }();
